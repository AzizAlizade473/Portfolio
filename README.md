# Aziz Alizade – Portfolio

Personal portfolio website built with React + Vite, Tailwind CSS, and Framer Motion.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 🛠️ Tech Stack

- **React** – UI library
- **Vite** – Build tool & dev server
- **Tailwind CSS** – Utility-first CSS framework
- **Framer Motion** – Animation library
- **React Icons** – Icon library

## 🚢 Deployment (Vercel)

1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → **New Project** → **Import from GitHub**
3. Select the `Portfolio` repository
4. Vercel auto-detects Vite → sets build command to `npm run build` and output to `dist`
5. Click **Deploy**
